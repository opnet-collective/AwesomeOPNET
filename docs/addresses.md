## Address Format Notes

OPNET addresses appear in two equivalent forms:

| Format | Example | Context |
|--------|---------|---------|
| **Bech32m** | `op1sq...` | Explorers, config files, human display |
| **32-byte hex** | `0x000...` | Contract calldata, event data, SDK calls |

To convert between formats, use `btc_getPublicKeyInfo` RPC method or any OPNET SDK.
